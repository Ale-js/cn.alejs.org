组件权限

介绍

在组件被定义时，你可以给组件添加权限（使用一个 permissions 参数来添加）：


其中，permissions 参数需要的是一个 Object 对象，一个 Ale组件对象 的函数对着一个布尔值，

当对值为 true 时，代表操作允许，当值为 false 时，代表操作不被允许（所有操作默认为 true 权限）。


以下为 permissions 参数支持的全部 Ale组件对象 的函数名称：

  • changeState
  • nextStates
  • addStates
  • removeStates
  • setStates
  • backStates
  • changeData
